# 图标配资

此目录需要包含以下尺寸的 PNG 图标：

## 必需的图标文件

- `icon-72.png` (72x72)
- `icon-96.png` (96x96)
- `icon-128.png` (128x128)
- `icon-144.png` (144x144)
- `icon-152.png` (152x152)
- `icon-180.png` (180x180)
- `icon-192.png` (192x192) ⭐ 最重要
- `icon-384.png` (384x384)
- `icon-512.png` (512x512) ⭐ 最重要

## 生成图标

打开项目根目录的 `generate-icons.html` 在浏览器中生成所有图标。

## 图标设计建议

- **颜色**：白色背景 (#FFFFFF)，黑色文字
- **内容**：显示"节气"二字
- **风格**：极简、中国风
- **装饰**：可添加24个小圆点代表24节气

## 在线图标生成器

如果不想手动生成，可以使用：

1. **favicon.io**: https://favicon.io/
2. **PWA Asset Generator**: https://www.pwabuilder.com/imageGenerator
3. **Canva**: 搜索 "PWA icon"

上传一个 512x512 的源图标，自动生成所有尺寸。
